# House Price Prediction

## Цель проекта
Построить модель машинного обучения для прогнозирования стоимости недвижимости на основе характеристик объектов.


## Данные
Используется датасет с параметрами недвижимости:
- площадь
- количество комнат
- расположение
- дополнительные характеристики

Источник: `HousePricePrediction.xlsx`


## Этапы работы

### 1. Анализ данных (EDA)
- исследование распределений признаков
- анализ корреляций

Файл: `EDA.ipynb`


### 2. Предобработка данных
- обработка пропусков
- кодирование категориальных признаков


### 3. Обучение модели
- разделение на train/test
- обучение модели линейной регрессии
- оценка качества модели

Файл: `preprocessing_to_model.ipynb`


## Используемые технологии
- Python
- Pandas
- NumPy
- scikit-learn
- Matplotlib / Seaborn


## Результаты

Модель позволяет прогнозировать стоимость недвижимости на основе входных параметров.

Используемые метрики:
- MAPE (Mean Absolute Percentage Error)
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- R² (коэффициент детерминации)


## Вывод

Модель выявляет зависимость цены от ключевых факторов:
- площади
- количества комнат
- других характеристик объекта

Проект демонстрирует полный цикл работы с данными:
EDA -> preprocessing -> обучение -> оценка модели


## Возможные улучшения
- использование более сложных моделей (Random Forest, Gradient Boosting)
- feature engineering
- подбор гиперпараметров
- добавление новых данных


## Практическое применение

Подход может использоваться для:
- оценки стоимости недвижимости  
- анализа рынка  
- построения рекомендательных систем
