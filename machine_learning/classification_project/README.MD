# Titanic - прогноз выживаемости пассажиров

## Цель
Построить модель машинного обучения для прогнозирования выживаемости пассажиров на основе характеристик (возраст, пол, класс каюты и т.д.).

## Этапы работы
- первичный анализ данных (EDA)
- обработка пропусков
- кодирование категориальных признаков
- масштабирование признаков
- обучение модели логистической регрессии
- оценка качества модели

## Используемые инструменты
- Python
- NumPy
- Pandas
- scikit-learn
- Matplotlib

## Результат
Точность модели (Accuracy): **0.76**

Модель правильно классифицирует ~76% пассажиров.  
Это означает, что она способна выявлять ключевые факторы выживаемости (например, пол, класс и возраст) и может использоваться для:
- базового анализа факторов риска  
- демонстрации подхода к задачам бинарной классификации  
- построения более сложных моделей на реальных данных  

## Вывод
Наиболее значимыми признаками оказались:
- пол (женщины выживали чаще)
- класс пассажира
- возраст
