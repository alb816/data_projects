## Описание

Проект демонстрирует навыки веб-парсинга, работы с MySQL и базового анализа данных в pandas
(агрегации, группировки, экспорт результатов в CSV).

## Функциональность

- Парсинг каталога товаров с веб-страницы
- Сохранение данных в MySQL
- Загрузка данных из БД в pandas DataFrame
- Расчёт средней цены товаров по категориям
- Экспорт агрегированных данных в CSV

## Используемые библиотеки

- Python
- requests / BeautifulSoup
- MySQL
- sqlalchemy
- pandas
